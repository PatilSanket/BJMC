# BJMC

<h2>Installation</h2>

1. create a new directory (Do Not create within frontend)
2. clone this repo. git clone https://github.com/PatilSanket/BJMC.git
3. git checkout master
4. npm install
5. Change username and password in server.js file 
5. node server
